# bioinf_lab1

Lab 1, CT Algorithms in Bioinformatics.

## Task 1

Напишите парсер данных формата FASTQ и заполните поля в тексте:

- Общее число прочтений в файле.
- Минимальная длина прочтения.
- Средняя длина прочтения, округлённая до целого числа.
- Максимальная длина прочтения.

Прохожусь по всем блокам, беру нужную строку, её длину. Далее считаю суммарную длину, минимум и максимум.

## Task 2

Рассчитайте GC-состав прочтений ([Rosalind: GC Content](https://rosalind.info/glossary/gc-content/)). В ответе введите число с точностью до двух знаков после запятой.

Считаю GC-состав - смотрю долю нуклеотидов G и C от общего числа.

## Task 3

Рассчитайте среднее значение качества по шкале Phred для всех прочтений для позиции 10. В ответе выведите целое число.

Прохожусь по блокам, беру строку качества, смотрю её 10-й символ, перевожу его по шкале Phred. Считаю среднее значение этой величины по всем прочтениям.
